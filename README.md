# Automatic_Servo_For_Light
宿舍语音/定时自动开关电灯装置

功能概述：
主要用stm32控制舵机，实现自动开关电灯按键，实现指标（不采用焊接，直接面包板）：
1.语音控制＋内部时钟定时，双模式信号控制舵机运动，最基本也是最重要的指标
2.平时99%的时间处于低功耗（休眠）模式，用干电池或可充电电池，保证装置可稳定使用一两年即可，第二重要
3.舵机机械传动部分，基本要求是平时不妨碍手动按开关以及机械结构相对比较经久耐用，且设计简单。扩展要求是区分两个开关按键
4.最后锦上添花的一点是用个盒子封装一下，安装时考虑直接放在柜子上或者热熔胶粘墙上，不过宿管可能有意见。至少舵机得粘墙上

器材：
1.stm32最小系统板
2.SG90舵机*2
3.5V稳压输出模块
4.LD3320语音模块
5.电源
